# Архитектура

## Обзор

Zapret2 Plus состоит из двух основных компонентов:

```
┌─────────────────────────────────────────────────────────────┐
│                    Magisk модуль                             │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐     │
│  │  nfqws2     │    │   Xray      │    │  Скрипты    │     │
│  │  (DPI)      │    │   (VPN)     │    │  (Управление)│     │
│  └─────────────┘    └─────────────┘    └─────────────┘     │
│         │                  │                  │             │
│         └──────────────────┼──────────────────┘             │
│                            │                               │
│                    ┌───────┴───────┐                       │
│                    │   iptables    │                       │
│                    │   NFQUEUE     │                       │
│                    └───────────────┘                       │
└─────────────────────────────────────────────────────────────┘
                            │
                            ▼
┌─────────────────────────────────────────────────────────────┐
│                   Android приложение (Kotlin)                │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐     │
│  │  UI слой    │    │  Слой данных │    │  Слой сервисов│    │
│  │  (Compose)  │    │  (Repository│    │  (NetworkMon) │    │
│  └─────────────┘    └─────────────┘    └─────────────┘     │
└─────────────────────────────────────────────────────────────┘
```

## Компоненты

### Magisk модуль

| Компонент | Путь | Описание |
|-----------|------|----------|
| nfqws2 | `zapret2/bin/*/nfqws2` | Движок обхода DPI |
| Xray | `zapret2/bin/*/xray` | VPN клиент (VLESS/VMess/SS/Trojan) |
| Скрипты | `zapret2/scripts/*.sh` | Скрипты управления и конфигурации |
| Конфиги | `zapret2/*.ini` | Файлы runtime-конфигурации |

### Android приложение

| Слой | Пакет | Описание |
|------|-------|----------|
| UI | `ui.screen.*` | Compose экраны |
| ViewModel | `viewmodel.*` | Управление состоянием |
| Data | `data.*` | Репозитории, менеджеры |
| Service | `service.*` | Фоновые сервисы |

## Поток данных

```
Действие пользователя → ViewModel → Shell команда → Magisk модуль → Результат
                            ↑                               │
                            └───────────────────────────────┘
                                     Обновление статуса
```

## Сетевой поток

```
┌─────────┐     ┌────────────┐     ┌───────────┐     ┌──────────┐
│ Приложение│────▶│  Zapret2   │────▶│ iptables   │────▶│ Интернет │
│ Трафик   │     │  (NFQUEUE) │     │  NAT       │     │          │
└─────────┘     └────────────┘     └───────────┘     └──────────┘
                      │
                      ▼ (опционально)
                ┌────────────┐
                │   Xray     │ VPN туннель
                └────────────┘
```

## Файлы состояния

| Файл | Назначение |
|------|-----------|
| `nfqws2.pid` | PID запущенного процесса |
| `xray.pid` | PID VPN процесса |
| `vpn-config.json` | Сгенерированная VPN конфигурация |
| `categories.ini` | Состояние включения/выключения категорий |
| `vpn-subs-raw.txt` | Сырые данные подписки |
